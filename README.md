# Mayank Bhardwaj - Portfolio

A premium, modern, and lightweight personal portfolio website for Mayank Bhardwaj, Senior Generative AI & Full Stack Engineer.

## Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Vanilla, CSS Variables, Flexbox/Grid, Glassmorphism)
- Vanilla JavaScript (Intersection Observers, Smooth Scrolling)

## Features
- **Dark Mode Aesthetic:** Premium glowing accents tailored for software engineering profiles.
- **Glassmorphism:** Elegant, modern glass cards for projects, skills, and experience timelines.
- **Performant Animations:** Intersection Observer API used for hardware-accelerated "reveal on scroll" transitions.
- **Fully Responsive:** Adapts seamlessly to all device sizes.
- **Zero Dependencies:** No heavy libraries, ensuring lightning-fast load times.

## How to Run Locally

You don't need any complex build tools to run this site locally.

1. Clone or download this repository.
2. Open your terminal and navigate to the project directory:
   ```bash
   cd path/to/Mayank-Bhardwaj-2703.github.io
   ```
3. Start a local development server. If you have Python installed, you can run:
   ```bash
   python3 -m http.server 8000
   ```
   Or if you use Node.js:
   ```bash
   npx serve .
   ```
4. Open your browser and navigate to `http://localhost:8000` (or the port specified by your server).

## Customization

- **Images:** Images are stored in the `assets/` and `assets/images/` directories. Replace them with your own keeping the same aspect ratios.
- **Colors:** You can change the primary and secondary glow colors by editing the CSS variables in the `:root` block of `styles.css`.
- **Content:** Update the text in `index.html`.
